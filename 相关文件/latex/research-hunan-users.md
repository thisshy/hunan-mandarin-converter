# 湖南项目受益人群数据审校记录

## 一、最终采用的数据口径

### 1. 本地用户基础

- 《2024年湖南省国民经济和社会发展统计公报》：
  - 2024年末湖南省常住人口 `6539万人`
  - 2024年末湖南省城镇人口 `4059万人`
  - 城镇化率 `62.07%`
- 采用原因：
  - 口径新，来源权威，适合用来说明项目面向本地用户时的基本盘较大。
- 来源：
  - https://tjj.hunan.gov.cn/hntj/tjgz/mthntj/202504/t20250421_33649403.html

### 2. 方言使用基础

- 湖南省政府门户网站《湖南方言概述》：
  - 湖南汉语方言主要包括 `湘方言、西南官话、赣方言、客家方言、湘南土话、乡话`
  - `湘方言使用人口约3085万人`
- 采用原因：
  - 能说明本项目不是面向一个很小的圈层，至少湘方言本身就有较大的使用基础。
- 使用时的谨慎表述：
  - 宜写成“仅湘方言使用人口就约3085万人”
  - 不宜直接写成“湖南全部方言使用人口为3085万人”
- 来源：
  - https://www.hunan.gov.cn/hnszf/jxxx/hxwh/jfy/201711/t20171111_4685261.html

### 3. 省内流动和跨地区交流需求

- 湖南省统计局《湖南区域经济发展中人口要素的分析》：
  - 湖南全省流动人口占当地常住人口比重 `12.02%`
  - 长株潭地区占比 `22.24%`
  - 湘南地区 `10.26%`
  - 洞庭湖地区 `9.10%`
  - 大湘西地区 `8.75%`
- 采用原因：
  - 适合说明省内跨地区流动较明显，核心城市群尤其需要跨方言沟通。
- 使用时的谨慎表述：
  - 这组数据来自第六次全国人口普查分析，适合做背景论证，不宜写成“最新现状”
- 来源：
  - https://tjj.hunan.gov.cn/m/jczx/201507/t20150717_3788521.html

### 4. 外来家庭和非本地用户场景

- 《2023年长沙市国民经济和社会发展统计公报》：
  - 2023年末长沙市常住人口 `1051.31万人`
  - 在校外来务工人员子女 `19.63万人`
- 采用原因：
  - 这组数据能比较直接地说明，核心城市中确实存在数量可观的非本地家庭，项目对外来用户不是“假设需求”。
- 使用时的谨慎表述：
  - 该数据为长沙市口径，适合说明核心城市场景，不宜直接推广为全省总量
- 来源：
  - https://tjj.hunan.gov.cn/hntj/tjfx/tjgb/szgb/zss_1/202405/t20240506_33292952.html

## 四、写入正文时的推荐判断

- 可直接写：
  - 项目的潜在使用者既包括本地用户，也包括非本地用户
  - 湖南本地用户基础较大
  - 长株潭等核心城市群存在更明显的跨地区交流需求
  - 长沙外来家庭在教育场景中具有现实使用需求
- 需避免的写法：
  - 把单一城市数据写成全省结论
  - 把湘方言人口等同于湖南全部方言人口
  - 把较早年份的流动人口数据写成“当前最新人数”
